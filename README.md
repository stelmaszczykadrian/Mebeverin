# Mebeverin

**The Mebeverin web application was created to present information about the drug and help users understand its use and effects.**

Features:
- Dynamic interface: SPA (Single Page Application).
- Responsiveness: Adapted to mobile and desktop devices.
- Hosting: Automatic deployment (CI/CD) using the Netlify platform.

[Mebeverin Link Here](https://mebeverin.netlify.app/)
